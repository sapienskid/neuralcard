# NeuralCard Roadmap

## Completed

**Core Features**
- FSRS algorithm integration with 4-button rating (Again/Hard/Good/Easy)
- Basic and Cloze card types
- Keyboard shortcuts (Space/Enter, 1-4, Escape)
- PouchDB sync with CouchDB support
- Folder-based deck organization
- Responsive dashboard with modern UI
- Full-width action buttons
- Mobile and small sidebar support
- Statistics page with activity charts

**Review Experience**
- Clean flashcard interface without keyboard hints
- Deck names in review/browse modal titles
- Custom study sessions with filters
- Card browser with navigation
- Progress tracking

## Planned Features

**Media Support**
- Audio playback in cards
- Text-to-speech integration
- Video embedding

**Card Management**
- Card flags (mark for review)
- Add notes to cards during review
- Bury cards until next day
- Suspend cards long-term
- Undo last review

**Analytics**
- Retention rate tracking
- Study streak counter
- Time spent per card
- Rating distribution charts
- Forgetting curve visualization
- Per-deck statistics

**Organization**
- Global tag system
- Tag browser and filtering
- Nested tags support
- Study by tag
- Bulk operations on cards

**Import/Export**
- Import from Anki
- Export decks with scheduling
- Deck sharing

**Mobile**
- Swipe gestures
- Quick add on mobile
- Full-screen review mode

**Performance**
- Lazy loading
- Image optimization
- Background sync

## Competitive Advantages

- Native Obsidian integration with bidirectional links
- Markdown-native cards with full Obsidian features
- Local-first with optional sync
- Self-hostable (PouchDB/CouchDB)
- Open source
